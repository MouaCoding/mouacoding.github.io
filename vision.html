Vision

Take a look at VISION now!

Vision is an Android app using Computer Vision. Some features will include: Face Detection, Character Recognition, Web Detection, Landmark detection, Explicit content. Some ideas: Dog Breed Identifier, Mood Identifier, Gender Change, Selfie filters

Used Bootstrap to design the front end website of “Vision” and AJAX to set up the front end web server communication (Back-End coming soon).
