Map Connect

Map Connect is a Realtime - Hyperlocal social media app on the Android Platform. It features a radar-ping of a set distance to calculate nearby events of the User's current location.

Users can post their events which is updated Real-Time on the Google Map, allowing other Users to see the NEARBY event and set if they want to go or not. They can search for nearby Points of Interests, set a map marker, and calculate how long it will take to get to the event.

Map Connect is the improved version of Right Here Right Now, utilizing much more of the Google Maps API, Geocoding API and Places API, as well as more efficient load times and better UI.

Uses: Android Studio, Google Maps API, Places API, Geocoding API, Firebase Database

Concepts: Firebase Management (Authentication, Database, Storage), Push Notifications and SMS texting, JSON Parsing, API Usage, Threads, UI/UX, Custom Lists and Adapters, Scrolls (Infinite Scrolling), Firebase Queries
