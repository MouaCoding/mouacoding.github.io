Stormi

Stormi is a Weather App for the Android platform. It features a background assigned to the weather conditions of the current User's location. It also features a 48 hour Forecast, and a daily forecast. The API used was Dark Sky Api, returning a JSON data file of a given latitude and longitude and Google Maps Geocoding to get the User's Current Location.

Uses: Android Studio, Google Maps Geocoding API, Dark Sky API, SQLite Database

Concepts: JSON Parsing, API usage, Threads, UI/UX, SQLite
